# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project allows two players to play turn by turn with automatic winner detection, draw detection, and a New Game option.

---

## 🚀 Features

- Two-player gameplay (X & O)
- Winner detection
- Draw detection
- New Game button
- Responsive user interface
- Clean and simple design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
